# 文档重构任务清单 v2

基于全库扫描与 Diataxis 框架分析，按「港美股券商帮助中心」用户心智模型重构。

## 目标结构（15 个顶层目录）

```
新手入门/ 开户与账户/ 入金/ 出金/ 资金划转与换汇/
股票交易/ 衍生品/ 新股认购/ 融资融券/ 基金与理财/
行情数据/ 资产与账单/ 活动与奖励/ 合规与税务/ 故障排查/
```

原 `App功能/` 目录整体消除，内容分散归入各业务目录。
