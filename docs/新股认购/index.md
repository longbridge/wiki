# 新股认购 / IPO

本章节介绍长桥的新股认购（IPO）服务，涵盖新股申购的完整流程、中签规则、资金冻结与扣款说明，以及暗盘交易的参与方式。

## IPO 参与时间线

从新股招股到正式交易，通常的流程是：

| 阶段 | 时间 | 你要做什么 |
|------|------|---------|
| **招股期** | 通常 1-3 天 | 在 App 新股版块提交认购申请，冻结资金 |
| **摇号与配售** | 招股结束后 | 系统根据规则决定你是否中签以及中签数量 |
| **暗盘交易**（可选） | 上市前一天 16:15-18:30 | 如果想提前交易，可在暗盘卖出或买入 |
| **正式上市** | 通常是工作日 | 股票正式在港交所挂牌交易，暗盘配资自动清结 |

**关键点**：新股申购是**可选机制**——你可以参与也可以不参与；但一旦参与，资金会被冻结直到摇号结束。
