# 看跌/看涨期权比率

看跌/看涨期权比率（Put/Call Ratio）是某一交易日所有看跌期权总成交量（或持仓量）与所有看涨期权总成交量（或持仓量）的比值。该指标反映市场看多或看空倾向，可结合历史变化趋势辅助判断市场情绪。

## 查看期权成交数据

点击顶部标签栏，可自由切换**总成交量**数据或**总持仓量**数据，同时展示当日数值与历史趋势曲线。

长按曲线图，可查看某一天的总成交量或总持仓量数据情况。

## 比率含义与应用

- **比率越小**：投资者更多地交易或持有看涨期权，表明市场看多倾向
- **比率越大**：投资者更多地交易或持有看跌期权，表明市场看空倾向

在实际应用中，Put/Call 比率最好结合总成交量/总持仓量数据一起分析。当总成交量/总持仓量数值同步增大时，以上结论越准确。

> 以上均为一般性规律分析，不代表市场运行必然结果，不构成任何投资建议。
