# 桌面端快捷键

桌面端全局和功能性快捷键、交易快捷键及自定义规则。

## 快捷键层级说明

- 全局快捷键：在任意界面下均可触发，无需聚焦特定组件
- 功能快捷键：需要鼠标焦点位于对应组件内才可触发

## 常用功能快捷键

| 快捷键 | 功能 |
|--------|------|
| Tab | 切换到下一个输入项 |
| Shift+Tab | 切换到上一个输入项 |
| Enter | 确认/提交 |
| Esc | 取消/关闭弹窗 |
| 方向键 ↑↓ | 上下移动选中项或调整数值 |

## 全局快捷键

| 功能 | Mac | Windows |
|------|-----|---------|
| 新开标签页 | Command+T | Ctrl+T |
| 切换到前一个标签页 | Command+[ | Ctrl+[ |
| 切换到后一个标签页 | Command+] | Ctrl+] |
| 组件快速切换（长按） | Option（长按） | Alt（长按） |
| 打开组件工具箱 | Command+E | Ctrl+E |
| 打开操作菜单 | Command+K | Ctrl+K |
| 打开交易弹窗 | Command+O | Ctrl+O |
| 向上翻页 | Command+↑ | Ctrl+↑ |
| 向下翻页 | Command+↓ | Ctrl+↓ |
| 快捷键说明页 | Command+/ | Ctrl+/ |
| 打开设置弹窗 | Command+, | Ctrl+, |

## 交易快捷键

### 全局通用（任意界面）

| 功能 | Mac | Windows |
|------|-----|---------|
| 买入弹窗 | Command+B | Ctrl+B |
| 卖出弹窗 | Command+S | Ctrl+S |
| 今日订单弹窗 | Command+D | Ctrl+D |

### 快速下单（焦点在交易组件时）

| 快捷键 | 功能 |
|--------|------|
| B | 买入 |
| S | 卖出 |
| Enter | 确认下单 |

## 快速撤单/改单快捷键

在订单列表组件中，选中订单后可使用：

| 快捷键 | 功能 |
|--------|------|
| Delete / Backspace | 撤销选中订单 |
| E | 改单 |

## 自定义快捷键规则

支持两类自定义快捷键：

- 单键：F1–F12 或标点符号键
- 组合键：Command/Ctrl + 数字、字母或标点符号

自定义设置入口：设置 > 快捷键。
