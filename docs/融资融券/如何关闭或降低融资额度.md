# 如何关闭或降低融资额度

长桥账户默认为融资账户（保证金账户）。如果您不需要使用杠杆交易，可以降低或关闭融资额度，以避免误操作产生融资利息。

## 降低融资额度

1. 打开长桥 App，进入「资产」页面
2. 找到「融资状态」或「我的融资额度」
3. 点击「调整额度」，将额度调低至您需要的金额
4. 确认提交，24 小时内生效

每 24 小时内最多申请 2 次（新加坡账户每 24 小时限 1 次）。

## 关闭融资额度

将融资额度调整为 0 即等同关闭融资。关闭后账户仍为保证金账户，但不可使用杠杆买入。

**前提条件**：关闭前需满足：
- 账户内无未平仓的融资持仓（即无结欠资金）
- 无待成交的融资买入委托

如有融资持仓，需先卖出持仓并还清欠款后再申请关闭。

## 确认是否有融资结欠

进入 App → 资产 → 现金页，查看「融资余额」或「结欠」字段：
- 如显示 0 或为空：无结欠，可直接关闭
- 如显示负数金额：账户有融资结欠，需先还清

还清融资的方式：入金至账户后，系统将在下一个结算日自动扣除结欠；或通过换汇操作偿还不同币种的结欠。

## 降低额度不影响既有持仓

- 降低额度不影响已有持仓，仅限制新增融资买入
- 持仓市值下跌导致维持保证金不足时，平台仍会发出追缴通知，与是否主动关闭额度无关
- 如需重新开启融资，按相同流程申请提高额度即可
