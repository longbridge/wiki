# 资产设置

资产设置用于自定义持仓页面的显示偏好，包括成本计算方式、持仓列表展示逻辑、期权和美股的价格引用规则。入口在「资产」→ 右上角设置图标，或「资产」→「全部功能」→「资产设置」。

> 该设置会影响「关注列表」「持仓」「订单」等多个页面的数据展示和布局。

## 股票名称显示

控制持仓列表中标的名称与代码的排列顺序：

- **股票名称在上**：默认，显示公司名称在前，代码在后
- **股票代码在上**：代码在前，名称在后，适合习惯用代码识别标的的用户

## 持仓合并显示

**同标的正股衍生品合并展示**：开启后，同一正股的股票和期权持仓合并显示为一行；关闭则分开独立列示。

## 持仓历史记录

| 设置项 | 说明 |
|--------|------|
| 显示已退市持仓 | 开启后显示已退市或摘牌标的的历史持仓记录 |
| 显示当日清仓标的 | 开启后当天已全部卖出的标的仍显示在持仓列表中（当日盈亏参考） |

## 成本计算方式

影响持仓盈亏的计算基准：

| 方式 | 说明 |
|------|------|
| 平均开仓成本 | 所有买入操作的加权平均价格，不含股息 |
| 摊薄成本 | 将股息再投资等因素纳入后的调整成本，更接近实际持仓成本 |

## 美股价格设置

| 设置项 | 说明 |
|--------|------|
| 美股使用盘前盘后价 | 开启后，盘前（04:00–09:30 ET）和盘后（16:00–20:00 ET）时段展示延伸交易价格 |
| 美股使用夜盘价 | 开启后，今日盈亏将在美东时间 20:00 重新计算，以夜盘最新价为基准 |

## 期权设置

| 设置项 | 说明 |
|--------|------|
| 期权使用延长时段价格 | 开启后期权估值引用盘前盘后价格，关闭则仅使用正式交易时段价格 |
| 期权表头设置 | 自定义期权持仓列表显示的列（如 Delta、隐含波动率、市值等） |

## 常见问题 Q&A

**Q：修改成本计算方式会影响历史记录吗？**

A：不会影响交易记录，但会重新计算持仓盈亏数字。建议确认计算方式后保持一致，避免反复切换导致盈亏数据混乱。

**Q：开启夜盘价后，今日盈亏为什么会在 20:00 跳变？**

A：美东时间 20:00 是夜盘开始时间，系统在此时以夜盘最新价重新计算当日盈亏，因此盈亏数字会在此刻刷新。这是正常现象，不影响持仓市值计算。

**Q：期权表头设置支持哪些列？**

A：常见可选列包括：市值、持仓盈亏、当日盈亏、Delta、Gamma、隐含波动率（IV）、到期日、行权价等。根据需要勾选显示。
