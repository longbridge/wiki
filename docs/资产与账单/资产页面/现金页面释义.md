# 现金页面释义

## 总金额

总金额为所有币种现金根据客户选择折算币种折算数值。按照实时汇率展示为对应币种，不是真实的货币兑换，该数据仅供参考。可能因货币兑换和跨币种展示之间的汇率差显示金额上少一些。

## 现金可提（最大可提）

所有账户内的现金/余额通按照实时汇率展示为对应币种可以用于交易/出金的资金。

客户开启余额通后会变为"最大可提"。

## 入金未到账

入金动作后在途还未到长桥账户内的资金。

## 余额通

点击可以进入余额通界面。

## 可用余额

客户可以用于交易的现金/余额通资金。

## 开启自动还款

开启后如果账户内有欠款/使用融资，会自动从余额通赎回或其他可用货币的资金跨货币进行货币兑换，以达到补齐欠款/还清融资。

特殊说明：存在香港公共假期但是美股正常开市的情况，在此期间余额通的资金不会自动赎回。如果当期交易了美股产生了欠款，会有计息，直到香港公共假期结束后基金公司自动赎回补平。以上情况产生的利息 APP 内不会显示欠款，可以查看客户结单。

## 结算中/待结算资金

买卖股票已成交但未交收的金额。港股的交收时间为 T+2，美股的交收时间为 T+1。

待结算金额可以用于同市场交易，但无法出金/货币兑换。

## 冻结中/冻结资金

包括交易当日产生的费用、待扣除利息、手动挂单冻结等静态资金冻结，以及因持有空头持仓产生的跟随持仓市值变化的平仓担保金等动态资金冻结。

一笔订单多次成交可能出现多次冻结费用的情况，但最后只会按照一笔订单实际扣费。

佣金会按照收费标准先冻结，但实际会按照免佣等活动减免扣费。
