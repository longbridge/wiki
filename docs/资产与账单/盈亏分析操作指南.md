# 盈亏分析操作指南

## 入口

以下两种方式均可进入盈亏分析页面：

方式一：点击资产总览页面的空白处，选择「盈亏分析」

方式二：长桥 App > 资产 > 全部功能 > 盈亏分析

## 功能概述

盈亏分析页提供全方位的交易盈亏回顾，主要包含以下内容：

累计盈亏：统计账户因交易或企业行动引起的盈亏总额

累计收益率：支持简单加权和时间加权两种计算方式

跑赢指数：与大盘指数的对比情况

每日收益：按日展示收益明细（收益日历）

个股收益：各标的盈亏明细，实时更新

资产变动：资金存取、转仓等资产变动记录

## 详细字段解释

各字段的详细定义、计算公式及常见问题，请参考盈亏页面释义文档。
