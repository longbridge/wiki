# 电汇入金（HK）

电汇仅支持跨境银行卡（非香港银行卡）转账入金，通过手机银行或网上银行操作。

操作步骤见 [如何通过电汇入金](./如何通过电汇入金.md)。

## 支持币种

港元（HKD）、美元（USD）

## 到账时间

预计 2-5 个交易日，具体取决于银行操作。

## 手续费

长桥不收取费用。
国际电汇入金到香港工银亚洲的收费为 60 HKD（或等值外币）。中转银行可能额外收费，请以实际到账金额为准。

## 支持的国家和地区

仅支持 FATF 成员国家/地区的跨境银行转账。请勿使用非 FATF 成员国家/地区的银行卡转账，入金会被退回且退款手续费由客户自行承担。

## 注意事项

- 暂不接受中国大陆银行发行的银行卡转账，款项将退回，产生的银行手续费不退还
- 转账银行账户名必须与证券账户名同名，不可使用他人账户转账
- 银行通知「已汇出」不等于长桥已收到款项，资金到达后需结算与审批
- 香港公众假期不处理汇款业务，请预留时间
- 不接受直接存入现金
