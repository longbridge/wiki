# FPS 转数快

## 介绍

FPS 转数快是香港金融管理局于 2018 年推出的快速支付系统，具有以下特色：

- 即时跨行转账
- 费用全免（大部分银行）
- 365×24×7 全天候转账服务
- 通过 FPS 识别码快速付款，无需使用手机号码或电邮地址登记

[官网操作指引](https://support.longbridgehk.com/topics/4jvueun/hk-fps?locale=zh-CN)

## 转账限额

长桥对 FPS 入金无任何限额，请留意银行端是否有转账限额。

## 操作流程

1. 长桥 App - 资产 - 存入资金 - 选择入金币种港元 - 选择入金银行卡 - FPS 转数快，复制 FPS ID
2. 登录网上银行或储值支付工具，选择 FPS 转数快功能进行转账汇款
3. 完成转账后，返回长桥 App - 资产 - 存入资金，选择存入币种及银行信息后选择 FPS 转数快，上传汇款转账凭证

注意：完成转账后请立即上传转账凭证和转账金额，否则会影响入金进度。

## 注意事项

- 转账使用的银行账户名必须与证券账户名同名，不可使用他人银行账户转账，否则退款费用由客户自行承担
- FPS 系统会将资金即时转入收款账户，但需经过人工审批后才能存入证券账户
- 银行通知「已汇出」不等于长桥已收到款项，资金到达后需进行结算与审批
- 香港节假日不处理汇款业务，请预留好汇款处理时间
- 不接受直接存入现金
