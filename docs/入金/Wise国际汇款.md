# Wise 国际汇款

Wise（原名 TransferWise）是一家国际汇款和货币转换服务平台，可以较低费用完成国际转账，将资金转入长桥新加坡账户。

操作步骤见 [如何通过 Wise 入金](./如何通过Wise入金.md)。

## 支持币种

美元（USD）、新加坡元（SGD）

## 到账时间

约 5 分钟至 3 个工作日，具体取决于入金币种。

## 手续费

长桥不收取费用。Wise 平台可能收取汇款手续费，以 Wise 实际收取为准。

## 收款账户信息

| 字段 | 内容 |
|------|------|
| 账户持有人 | LONG BRIDGE SEC |
| 银行账号 | 0725885663 |
| 银行名称 | DBS BANK LTD |
| SWIFT/BIC | DBSSSGSGXXX |
| 汇款备注 | 填写证券账户号（如 SG10039100），**必填** |

## 注意事项

- 首次通过 Wise 入金后，可以入金但不能通过 Wise 出金
- 不接受第三方汇款（非本人账户），退回资金产生的手续费由客户自行承担
- 转账账户名必须与长桥证券账户名同名
