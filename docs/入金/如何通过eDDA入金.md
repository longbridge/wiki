# 如何通过 eDDA 入金（HK）

eDDA 使用前需先完成授权。费率与常见错误说明见 [eDDA 电子授权扣账说明](./eDDA电子授权扣账.md)。

## 第一步：完成 eDDA 授权

### 方式一：在长桥 App 内发起（推荐）

1. 打开长桥 App，进入 **资产 → 存入资金 → 存入港元 → 香港银行卡**
2. 在银行卡列表中选择具体银行卡 → **eDDA 入金** → 申请授权（银行名称右边有 eDDA 标签的表示支持）
3. 确认银行授权信息（卡号、姓名），点击「确认授权」
4. 等待银行授权结果，授权成功后会收到推送消息和邮件通知

> 开户证件类型、姓名排序必须与银行开户时一致，否则授权失败。大部分银行约 5 分钟出结果，个别银行需 2-14 个工作日。

### 方式二：在银行端发起

花旗银行、东亚银行、招商银行需在银行网页端发起 eDDA 授权，授权成功后再在长桥 App 发起入金。在长桥 App 内多次授权失败时也可尝试此方式。

各银行详细操作步骤请在长桥 App 入金页面查看（汇丰、恒生、中银香港、工银亚洲、招商银行等）。

## 第二步：发起入金

授权成功后，在长桥 App 的 eDDA 入金页面点击「立即入金」，输入金额即可。无需上传入金凭证。

## 解除授权与删除银行卡

- 解除 eDDA 授权需到对应银行网银端操作，App 端不支持解除
- 需先解除 eDDA 授权，才能在 App 删除该银行卡
- 如在长桥侧解绑后再次绑定出现「重复授权」报错，通常是银行端未成功解除，需联系银行操作
