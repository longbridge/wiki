# 如何通过 ATM / 柜台 / 支票入金（HK）

收款账户信息见 [ATM 与柜台支票说明](./ATM与柜台支票.md)。

## 方式一：ATM / 柜台转账

1. 获取长桥收款账户信息（见说明页）
2. 前往香港银行 ATM 或柜台，将资金转账至长桥收款账户
3. 保留汇款凭证截图（ATM 打印凭条或柜台回执）
4. 打开长桥 App，进入 **资产 → 存入资金 → 选择币种及银行信息 → ATM / 柜台转账**，上传汇款凭证

> 完成转账后请立即上传凭证，否则会影响入金进度。

## 方式二：支票转账

1. 填写支票
   - 正面：收款人填写 **Long Bridge HK Limited**
   - 反面：填写 **长桥综合账户（你的长桥账户号码）**，如「长桥综合账户（H10057829）」
   - 填写完成后拍摄支票正面和反面照片
2. 将支票递交至银行柜台或存支票机
3. 保留转账凭证截图
4. 打开长桥 App，进入 **资产 → 存入资金 → 选择币种及银行信息 → 支票转账**，上传支票正面截图、反面截图和转账凭证

> 完成后请立即上传所有凭证，否则会影响入金进度。
