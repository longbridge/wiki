# 如何通过 FPS 转数快入金

入金前请确认：银行账户名与长桥证券账户名**同名**，且已准备好收款 FPS ID（见 [FPS 转数快说明](./FPS转数快.md)）。

## 操作步骤

1. 打开长桥 App，进入 **资产 → 存入资金**，选择入金币种「港元」，选择银行卡，点击「FPS 转数快」，复制 FPS ID
2. 登录网上银行或储值支付工具，选择 FPS 转数快功能，向复制的 FPS ID 发起转账
3. 转账完成后，立即返回长桥 App **资产 → 存入资金**，选择币种及银行信息，选择「FPS 转数快」，上传汇款转账凭证

> 凭证必须在转账后**立即上传**，否则会影响入金进度。

## 各银行操作指引

长桥 App 入金页面提供以下银行的详细操作步骤：

- 汇丰银行
- 工银亚洲
- 招行香港
- 众安银行

如需查看，请在 App 入金页面选择对应银行后查看指引。
