# eDDA 电子授权扣账

eDDA 快捷入金是香港金管局推出的 FPS 转数快系统增值服务，支持授权长桥证券从指定银行账户代扣资金存入证券账户，无需去银行操作，无需上传转账凭证。

特点：安全（双方系统核实身份）、极速（5×24 入金，5 分钟极速到账）、免费、支持恒生/汇丰等多家香港银行。

操作步骤见 [如何通过 eDDA 入金](./如何通过eDDA入金.md)。

## 支持币种

仅支持港元（HKD）

## 到账时间

| 发起入金时间 | 预计到账时间 |
|------------|------------|
| 港股交易日 | 5 分钟内 |
| 其他时段 | 下个港股交易日 9:40 前 |

如遇港股休市日，所有到账时间顺延。

## 手续费

免费。但账户余额不足时，部分银行（如汇丰银行）可能扣手续费，转账前需确保账户资产充足。

## 常见授权失败原因

| 错误信息 | 原因 |
|---------|------|
| Debtor account name / ID not matched | 证件类型或姓名格式问题，字母需大写 |
| Invalid debtor account number | 账号错误、加了银行代码、或使用了分币种账号 |
| Mandate identification already exists | 银行端与长桥 App 端同时发起授权 |
| Debtor account not found | 账号有误 |
| Rejected by Other bank | 银行卡账户类型问题（如绑定了美元账户类型） |
| Invalid Payer ID | 银行卡证件信息与开户信息不一致 |

如排查完所有原因仍无法授权，可联系银行重置 eDDA 后再次尝试。华侨银行一般需 2 个工作日处理，如长时间未出结果，可能是银行代码选错，可删除后重新发起。

## 注意事项

- 转账银行账户名必须与长桥证券账户名同名，不可使用他人账户转账
- 银行通知「已汇出」不代表长桥已收到款项，资金到达后需结算与审批
- 香港节假日期间不处理汇款业务，请预留时间
- eDDA 不支持出金
