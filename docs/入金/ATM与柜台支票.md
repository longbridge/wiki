# ATM / 柜台支票入金（HK）

通过香港银行 ATM、银行柜台或支票将资金转账至长桥收款账户，转账完成后需上传汇款凭证。

操作步骤见 [如何通过 ATM / 柜台 / 支票入金](./如何通过ATM柜台和支票入金.md)。

## 支持币种

港元（HKD）、美元（USD）

## 到账时间

| 方式 | 预计到账时间 |
|------|------------|
| ATM / 柜台同行转账 | 2 小时内 |
| ATM / 柜台跨行转账 | 1-3 个工作日 |
| 支票转账 | 约 2 个工作日（银行需处理支票入账） |

## 手续费

长桥不收取费用，具体费用以银行实际收取为准。

## 注意事项

- 仅适用于香港银行柜台及 ATM
- 不建议支票转账至工商银行（亚洲）收款账户，核对客户资料需额外时间，入账速度较慢
- 仅支持香港银行支票，不支持香港银行本票
- 转账银行账户名必须与证券账户名同名，不可使用他人账户转账
- 银行通知「已汇出」不等于长桥已收到款项，资金到达后需结算与审批
- 香港公众假期不处理汇款业务，请预留时间
- 不接受直接存入现金
