# 网银与电汇入金（SG）

通过手机银行或网上银行将资金转账至长桥新加坡收款账户的入金方式，新加坡本地转账和跨境电汇均通过此方式操作，转账完成后需上传汇款凭证。

操作步骤见 [如何通过网银和电汇入金](./如何通过网银和电汇入金.md)。

## 支持币种

新加坡元（SGD）、美元（USD）、港元（HKD）

## 到账时间

预计 1-3 个交易日

## 手续费

长桥证券及其代理银行（DBS）不收取任何费用。汇款银行和中转银行可能收取费用（约 200 港元或 25 美元）。

## 收款银行信息

**新加坡地区（FAST / 本地转账）及其他地区（国际电汇 SWIFT）：**

| 字段 | 内容 |
|------|------|
| 收款人姓名 | LONG BRIDGE SEC |
| 收款人账号 | 0725885663 |
| 收款银行 | DBS BANK LTD |
| SWIFT 代码 | DBSSSGSGXXX |
| 附言（汇款备注） | 填写证券账户号（如 SG10039100），**必填** |

费用承担方：SHA（各环节各自扣除）

## 特殊情况说明

**iFAST Global Bank（奕丰环球银行）**：属于跨境电汇，因使用第三方支付平台，收款行可能无法解析付款方信息，需在长桥上传 iFAST 开户证明及汇款流水。

**IBAN**：海外银行卡可在银行账户处直接输入 IBAN Code。不了解 IBAN 的可查询银行结单获取。

**三位数银行代码**：主要用于香港地区，海外银行卡没有银行代码时可填写 000。

**中转行显示问题**：跨境电汇资金可能显示为中转行账户，如遇此情况请联系客服。

## 注意事项

- 转账银行账户名必须与长桥证券账户名同名，不可使用他人或联名账户转账
- 银行通知「已汇出」不等于长桥已收到款项，资金到达后需结算与审批
- 节假日期间不处理汇款业务，请预留时间
- 不接受直接存入现金
