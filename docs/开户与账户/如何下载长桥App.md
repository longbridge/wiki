## 苹果（iOS）

可通过 App Store 更新/安装长桥 App，包括自动更新、手动更新/安装两种方式。

> **中国大陆用户注意**：若长桥 App 在中国大陆 App Store 无法搜索到，请使用下方「手动更新/安装」通过【已购项目】操作，无需切换 Apple ID 区域。

### 方式 1：自动更新

打开【设置】-【App Store】-【App 更新】，打开其中的【App 更新】开关，即可实现长桥 App 的自动更新。

注意：
1. 该方式下，您手机上的所有 App 均为自动更新；
2. 若您误删了长桥 App，请参考下面的「方式 2：手动更新/安装」进行安装。

### 方式 2：手动更新/安装

打开【App Store】，点击右上角头像进入【账户】页面，**先在【账户】页面下拉刷新**（若不刷新，可能后面步骤中看不到【更新】按钮，因此请务必下拉刷新），再点击进入【App】（较新 iOS 版本）或【已购项目】（较旧 iOS 版本），在【全部】中搜索"长桥"，找到长桥 App。

1. 若长桥 App 有更新，则可以看到【更新】按钮，点击即可更新；
2. 若误删了长桥 App，则可以看到【下载】按钮，点击可下载安装；
3. 若找不到长桥 App，请参考下面的「常见问题」。

### 常见问题：在【App】/【已购项目】中找不到长桥

可能原因及解决方法如下：

1. **App 被隐藏**：打开【App Store】→ 右上角头像 → 账户 → 顶部账户名 → 【账户设置】→ 【隐藏的已购项目】，找到"长桥"后点击【解除隐藏】，再按方式 2 操作；
2. **Apple ID 不匹配**：当前登录的 Apple ID 不是当初下载长桥时使用的账号。打开【设置】→ 顶部账户名 → 拉到底部【退出登录】，换回之前的 Apple ID 重新登录，再按方式 2 操作。
