# 账户常见问题

二步验证、短信验证码接收、休眠账户定义及激活方法。

## 什么是二步验证

当进行交易、查看账户信息等与资产相关的操作时，需要输入交易密码。二步验证功能启用后，在校验完交易密码后，App 会通过安全验证或手机验证码进一步确认身份。

建议启用二步验证功能以保护账户安全。

## 收不到短信验证码怎么办

若无法接收短信验证码，请按以下步骤排查：

1. 检查验证码短信是否被手机拦截（查看垃圾短信或拦截记录）
2. 检查网络环境是否良好
3. 检查收件箱是否已满

如排查后仍无法收到，可尝试使用密码登录，或联系官方客服。

## 什么是休眠账户

休眠账户是指长期未发生交易活动的账户。需同时满足以下条件：

- 账户近 3 年无任何主动操作记录（包括证券交易、新股申购、基金认购、资金存入/提取、股票存入/提取等）
- 账户资产净值低于等值 100 港币

休眠账户将暂停交易、出金及股票转出功能。如需重新激活，请通过以下方式联系长桥客服办理：

- 在线客服
- 客服热线
- 客服邮箱
