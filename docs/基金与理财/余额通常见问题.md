# 余额通常见问题

场景：余额通 - 常见问题、手动赎回、收益计算、新股认购、转仓、出金

用户可能问：余额通有什么特色、手动赎回的钱能马上买股票吗、余额通能用来换汇吗、余额通可以支付转仓费吗、关闭余额通后多久到账

余额通五大特色：
- 随心炒股：余额通资金可直接买入股票，无需提前赎回
- 闲钱增值：闲置资金自动投入货币基金，收益高于活期
- 一劳永逸：开通后全自动运作，无需手动操作
- 货基打底：以货币基金为底层资产，相对稳健
- 低起购：最低 0.01 起投

手动赎回规则：

香港余额通：新版 8:30 前赎回当日处理，旧版 9:10 前赎回当日处理；超过截止时间则顺延至下一交易日。

- 手动赎回后资金可立即用于股票交易（T+0 可用）
- 手动赎回后 3 个自然日内不会触发自动转入

货币兑换和出金：
- 需先手动赎回并等待资金到账，之后才能进行换汇或出金操作
- 不支持在余额通赎回过程中直接换汇或出金

新股认购限制：
- 若账户资金在余额通中，需在新股截止认购日的前一交易日 9:30 前完成申购（确保资金及时到位）
- 建议提前确认资金状态，避免因赎回时间不足导致认购失败

转仓费用：
- 暂不支持使用余额通资金直接支付转仓费用，需提前将资金赎回至普通账户

- 新版香港余额通：每天 8:30 计算当日收益

关闭余额通到账时间：
- 关闭后系统在最近交易日发起赎回，最晚 T+1 个交易日到账
