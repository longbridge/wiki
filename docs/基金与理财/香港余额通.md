# 香港余额通介绍与使用规则

场景：余额通 - 余额通、货币基金、闲置资金、现金增值

用户可能问：香港余额通是什么、余额通支持哪些币种、余额通怎么用、余额通和普通活期有什么区别、新版余额通和旧版有什么区别

余额通是长桥为港股账户提供的闲置资金增值服务，将账户内闲置港元（HKD）或美元（USD）自动投入货币基金，获取高于活期存款的收益，同时不影响股票交易。

主要特色：
- 股票交易不受限制：余额通资金可直接用于买入股票，无需手动赎回
- 每日自动投资：闲置资金每天固定时间自动转入货币基金
- 收益高于活期：货币基金收益通常优于银行活期利率
- 低起投门槛：0.01 起投

新版余额通支持每个币种选择多只货币基金：港元（HKD）和美元（USD）各有三只基金可选。

申购管理：
- 可在设置中修改下次自动申购的基金
- 支持手动申购，随时将闲置资金主动转入

自动转入/赎回机制：
- 开通并签署授权后，每个交易日固定时间自动将闲置资金转入所选货币基金
- 买股票、换汇等扣款操作触发时，系统自动从货币基金赎回所需金额

手动赎回规则：
- 手动赎回后，3 个自然日内不会自动转入
- 赎回资金可立即用于股票交易
- 货币兑换和出金需等赎回到账后才能操作

交易时间：新版余额通每天 8:30 自动转入和转出；旧版每天 9:10 自动转入转出。

关闭余额通：
- 关闭后，系统在最近交易日赎回持仓，最晚 T+1 到账
- 关闭后原余额通持仓将转入普通基金账户
- 关闭后购买力立即下降，若账户有融资可能触发追缴保证金，请注意风险
