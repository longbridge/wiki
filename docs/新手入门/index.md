# 新手入门

本系列教程帮助你完成从零到第一笔交易的完整过程。完成三个步骤后，你将拥有一个有资金的长桥账户、第一个港股持仓，并掌握在市场变化时如何跟踪股价的方法。

## 学习路线

这些教程遵循**从准备到行动的自然流程**：

1. **准备资金** — 没有资金无法交易，所以首先入金
2. **完成交易** — 资金到账后立即执行你的第一笔买入
3. **持续监控** — 交易完成后，学会关注股价变化

**总用时**：18 分钟操作 + 入金处理时间（通常 2 小时）

- [设置股价提醒](/新手入门/设置股价提醒) — 设置目标价，触达时收到手机推送

## 完成后的下一步

掌握了基础交易后，你可以：
- 探索[更多交易订单类型](/股票交易/订单类型/)，如止盈止损、长期单等
- 了解[港股交易的规则与费用](/股票交易/)，为更复杂的策略做准备
- 学习[如何管理持仓](/资产与账单/)，监控盈亏和账户风险
