# 合规与税务

平台开户资质、隐私政策，以及美股税表、FATCA、CRS 等税务合规说明。

## 你需要了解这部分的场景

- **刚开户**：确认你所在国家/地区是否能使用长桥，以及个人数据如何被保护
- **投资美股**：美股涉及复杂的税务规定（如预扣税），需要理解 W-8BEN 表、FATCA、税务条约
- **大额交易**：某些国家对跨境资本流动有报告要求（AEOI/CRS），需要了解义务和流程
- **年度报税**：需要从长桥获取税表或交易数据进行申报
