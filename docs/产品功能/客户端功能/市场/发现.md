# 发现

发现是市场 Tab 内的综合标的探索与工具入口页，入口在底部导航「市场」→「发现」子 Tab。

## 快捷工具栏

页面顶部横向滚动的功能图标：

| 工具     | 说明            |
|--------|---------------|
| 美股 ETF | ETF 筛选与行情列表   |
| 余额通    | 快速进入余额通管理页    |
| 股票 VS  | 两只股票横向对比分析    |
| 新股申购   | 港/美股 IPO 认购入口 |
| 更多     | 展开全部工具        |

## 热股异动事件

实时监测并展示当日有异动的股票，包括：

- 跌幅达 X%
- 波动超 20 日均值
- 成交量突然放大等异动类型

点击卡片进入标的详情页查看异动原因和相关资讯。

## 官方股单

平台精选主题组合，覆盖热门投资方向：

- AI 算力核心、AI 软件应用
- 港股机器人概念、港股光伏、港股创新药
- SpaceX 相关、OpenClaw 相关
- 内存与存储、韩国市场相关股

点击股单可查看成分股及整体表现，支持一键关注。

## 产业链视图

可视化展示特定行业的上中下游企业结构：

- **纵向分层**：基础层 / 技术层 / 应用层
- **横向分类**：基础设备 / 云服务提供商 / 终端应用等

自选列表中涉及该产业链的标的会在图中高亮标注。

## 财报预告

列出即将发布财报的公司、预计发布日期及市场预期数据，便于提前关注和规划交易。

## 策略选股 / 选股器

通过多维度条件组合筛选符合投资逻辑的股票：

- 股息率（TTM）≥ 10%
- 高成长公司
- 高股息蓝筹股
- 自定义近 N 月涨跌幅范围等

## 经济数据日历

显示全球主要宏观经济指标的发布时间和前值，如非农就业、CPI、央行利率决议等，帮助把握市场节奏。

## 常见问题 Q&A

**Q：热股异动是实时的吗？**

A：异动数据在交易时段内持续更新（页面显示更新时间戳），非实时推送，有短暂延迟。

**Q：产业链视图覆盖哪些行业？**

A：覆盖 AI、半导体、新能源、消费等主要行业，平台会根据市场热点不定期新增产业链图谱。

**Q：选股器筛选结果可以保存吗？**

A：目前支持保存为自定义策略，下次进入时可直接调用已保存的筛选条件。

**Q：财报预告的数据来源是哪里？**

A：来自彭博、路孚特等数据供应商提供的市场一致预期数据，实际财报发布后系统会自动与预期对比标注超预期/不及预期。
