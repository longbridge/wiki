# 兑换商城

兑换商城是使用任务币换取实物礼品和平台权益的专区。

## 页面入口

「我的」→「兑换商城」，或任务中心首页 Banner 直达。

## 任务币余额

页面顶部显示当前任务币总余额，可点击「任务币详情」查看收支记录，点击「赚任务币」跳转任务中心。

## 库存更新

全部商品库存**每周一 17:00** 统一更新，库存有限，先到先得。

## 商品分类

### 实物礼品

包含数码配件、品牌联名周边、长桥定制礼品等。兑换后需填写收货地址，由平台安排发货。

### 行情权益

包含港股、美股等市场的高级行情订阅服务，以及 Open API 行情授权，适合有实时行情需求的用户。

### 现金券 / 抵扣卡

包含平台手续费抵扣卡、期权现金券等，可用于抵扣相应交易费用。

### 投资课程

包含各类投资主题课程，兑换后可在 App 内直接学习。

### 活动道具

包含可用于猜涨跌等互动活动的功能性道具（如倍数卡等）。

## 兑换流程

1. 浏览商城选择心仪商品
2. 确认任务币余额是否充足
3. 进入商品详情页，确认规格和说明
4. 点击「立即兑换」并完成确认
5. 实物礼品填写收货地址；虚拟权益直接到账

## 注意事项

- 标注「限时折扣」的商品在折扣期结束后将恢复原价
- 实物礼品兑换后需填写真实有效的收货地址
- 虚拟权益（行情卡、现金券等）到账后可在「奖励记录」页查看
- 任务币一经兑换不支持退还
