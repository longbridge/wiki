# 任务中心

任务中心是长桥 App 的积分任务系统，通过完成各类日常任务赚取「任务币」，再用任务币在兑换商城换取礼品或权益。

## 页面入口

「我的」→「任务中心」

## 任务币

任务币是任务中心专属的积分货币，完成任务后系统自动发放。

- 可在兑换商城用任务币兑换实物礼品、行情权益、现金券等
- 点击页面顶部「任务币详情」可查看收支明细

## 任务分类

| 任务分类 | 说明                   |
|------|----------------------|
| 每日任务 | 每天可完成，次日重置，包含签到、猜涨跌等 |
| 每周任务 | 每周重置，包含交易、社区互动等操作类任务 |
| 成长任务 | 一次性里程碑任务，完成后不再重复出现   |
| 月度挑战 | 当月限时挑战任务，月底截止        |

## 每日签到

- 每天签到可获得任务币奖励
- 签到页展示当月日历，可查看连续签到记录
- 连续签到达到特定天数里程碑时可获得额外奖励
- 当天已签到时提示「您今天已签到，明天再来吧」

## 猜涨跌

猜涨跌是每日任务之一，每天开放对特定股票涨跌方向的预测：

- 选择「猜涨」或「猜跌」参与当日预测
- 页面实时显示猜涨和猜跌的用户比例
- 设有任务币奖池，结果揭晓后猜对方向的用户按比例瓜分
- 页面显示开奖倒计时

## 月度挑战

月度挑战在普通每日/每周任务之外提供额外奖励机会，完成指定数量的挑战任务可获得当月特别奖励。具体任务内容和奖励每月更新。
