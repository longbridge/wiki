# 专享特权

「专享特权」展示账户当前已解锁的专属权益，以及可通过达成条件解锁的额外权益。入口在「我的」→ 蓝色专享特权卡片，或「我的」→「专享特权」。

## 交易权益

### 终生免佣

- **港股终生免佣**：港股股票、ETF等交易佣金永久减免为 0
- **美股终生免佣**：美股股票、ETF等交易佣金永久减免为 0

> 终生免佣仅免除长桥证券自身佣金，香港政府印花税、交易所规费等第三方代收费用仍按规定收取。

## 行情权益

| 权益    | 说明                            |
|-------|-------------------------------|
| 港股 L2 | 港交所 Level 2 实时十档买卖盘深度行情       |
| 全美行情  | 全美股市场（NYSE、NASDAQ、AMEX 等）实时行情 |
| 深度摆盘  | 更深层次的订单簿数据                    |

**港股 L2 地域限制**：应港交所要求，L2 行情区分地域和设备类型。中国大陆移动端授权仅绑定单一设备，不可在多台设备同时使用；香港及其他地区无此限制。

## 解锁方式

部分权益需完成资产或交易要求后，在「兑换商城」兑换激活：

1. 在「专享特权」页面查看待解锁权益的具体条件
2. 完成对应条件（如达到指定交易量或持仓金额）
3. 前往「我的」→「兑换商城」完成兑换

## 累计已省

「我的」主页专享特权卡片上显示**累计已省金额**，统计通过终生免佣等权益节省的交易费用总额，可作为权益实际价值的参考。

## 邀好友共享

点击专享特权页面内的「邀好友共享」，生成专属邀请链接。好友通过链接注册并满足条件后，双方可同时获得额外权益奖励。

## 常见问题 Q&A

**Q：终生免佣是永久的吗？会不会某天取消？**

A：「终生」是指在长桥平台有效运营期间持续有效，不设到期时间。若平台规则有重大调整，会提前通知。

**Q：港股 L2 行情换手机后还能用吗？**

A：中国大陆用户的 L2 授权绑定设备，换手机后需重新激活绑定。进入「专享特权」或「行情权限」页面按提示操作，或联系客服协助处理。

**Q：「待解锁」的权益达成条件后会自动激活吗？**

A：不会自动激活，需要主动前往「兑换商城」完成兑换操作。建议达成条件后及时兑换，避免遗忘。

**Q：专享特权的权益可以转让给他人吗？**

A：不可以。所有权益绑定到账户本人，不支持转让、赠送或出售。
