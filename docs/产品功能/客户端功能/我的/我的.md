# 我的

「我的」是 App 底部导航最右侧的个人中心，汇聚账户信息、权益入口、功能快捷方式和设置入口。

## 用户信息区

页面顶部展示：

- 头像、昵称、认证标签（如「长期价值投资者」）
- **登录 ID**（格式如 `32*****88`），可用于 Login ID 登录方式
- 关注数 / 关注者数 / 我的收藏数 / 浏览记录数，点击各项可跳转对应列表

## 专享特权卡片

蓝色卡片，显示：

- **累计已省**：通过免佣等权益节省的交易费用合计
- 权益快捷图标（港股免佣、美股免佣、行情、融资利率等），点击跳转对应权益详情
- **待解锁提示**（如「2 项待解锁」），点击查看可解锁条件

## 常见问题 Q&A

**Q：「累计已省」金额怎么计算？**

A：以正常费率计算本应缴纳的交易费用，减去通过免佣权益实际支付的金额（通常为 0），两者差值即为累计已省金额。

**Q：登录 ID 在哪里查看？**

A：「我的」主页用户名下方直接显示，格式为纯数字（如 `32***88`）。

**Q：「我的」页面的关注数点进去能看什么？**

A：点击「关注」可查看你关注的所有用户列表；点击「关注者」查看关注你的用户。
