# 行情商店

行情商店是长桥的行情数据订阅入口，支持按市场分类浏览和购买各类高级行情套餐。

## 页面入口

「我的」→「行情商店」

## 市场分类

行情商店按市场分为三个标签页：**美股**、**港股**、**新加坡**。

## 我的行情

点击右上角「我的行情」查看当前账户的行情权限状态，支持按市场（全部 / 美股 / 港股 / 沪深 / 新加坡）筛选。

### 已开通行情

展示当前已订阅或推广期内免费开通的行情权限，每条显示：

- 行情产品名称和说明
- 所属市场
- 到期时间或「推广期免费」标注

### 未开通行情

展示尚未订阅的可用行情产品，点击可跳转至购买页。

> 部分行情产品（如港股 LV2 行情）受地域限制，仅支持在特定地区使用，购买前请确认是否符合使用条件。
