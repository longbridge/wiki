# 窝轮

[HK 帮助中心](https://support.longbridgehk.com/topics/hk-trade/3et7xd6)

## 什么是窝轮

窝轮本质上是一种期权，相当于持有一种虚拟的权利。以持有人的权利而言，分为认购证和认沽证。

### 认购证（Call）

赋予持有人以行使价在特定期限内购买一定数量相关资产的权利（非责任）。看好相关资产后市时买入。

### 认沽证（Put）

赋予持有人以行使价在特定期限内出售一定数量相关资产的权利（非责任）。看空相关资产后市时买入。

## 交易规则

### 交易时段

窝轮在港股持续交易时段交易，与港股交易时间一致。

### 交易入口

股票详情页进入轮证页面，点击对应窝轮，选择「买入」或「卖出」，输入价格和数量下单。

### 支持订单类型

限价单、市价单、条件单、策略单及长期单。

### 最小交易单位

按照窝轮自身的手数限制。

### 交收规则

T+0 交易，T+2 交收，均为现金结算。

## 简称含义

以 19197「华能法巴五零一购（@EC）」为例：

| 部分 | 含义 |
|------|------|
| 华能 | 相关资产为华能国际（902） |
| 法巴 | 发行商为 BNP |
| 五零一 | 到期日为 2005 年 1 月（具体日期需查看 APP 轮证行情页） |
| 购 | 认购证（认沽证显示「沽」） |
| @ | 现金结算（* = 现货结算，$ = 现金或现货结算） |
| E | 欧式备兑证（European） |
| C | 认购证 Call（认沽证显示 P） |

### 月份表示规则

1-10 月以中文数字（一、二…十）代表，11 月以「甲」代表，12 月以「乙」代表。同一发行商同一资产同一到期月再发行的以系列编号 A、B、C 区分。

## 轮证结算

### 剩余价值计算

- 认购证：剩余价值 =（结算价 - 行使价）÷ 换股比率。仅到期时结算价 > 行使价（价内）时有效，否则为零
- 认沽证：剩余价值 =（行使价 - 结算价）÷ 换股比率。仅到期时结算价 < 行使价（价内）时有效，否则为零

### 结算价

- 个股窝轮：到期日前（不含到期日）5 个交易日的正股平均收市价
- 指数窝轮：以联交所公布的对应期货合约最终结算价为准（如恒指窝轮使用恒指期货的 EAS，即到期日当天的 5 分钟平均价）

## 关键行情字段

### 街货量与街货比

- 街货量：市场上由其他投资者持有的轮证数量
- 街货比：投资者持有的轮证占总发行量的比例

### 换股价与换股比率

- 换股比率：多少单位的轮证可以换 1 股正股
- 换股价：兑换时对应正股的价格
