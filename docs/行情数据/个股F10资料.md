# 个股F10资料

## 介绍

F10 是个股基本面资料的统称，在长桥 App 中通过个股详情页的「全景」标签查看。提供上市公司的公司概况、财务数据、股东信息、分红记录等全方位基本面信息，帮助投资者深入了解和分析公司。

入口：长桥 App - 个股详情页 - 全景

## 公司概况

- 公司简介：公司主营业务、成立时间、注册地、上市日期等基本信息
- 行业分类：所属行业板块
- 管理层信息：高管团队和董事会成员
- 联系方式：公司地址、官网、投资者关系联系方式

## 财务数据

长桥 App 提供三大财务报表的深度分析，涵盖 30+ 核心财务指标。

### 利润表

展示公司营业收入、净利润、毛利率、净利率等盈利指标，可查看季度和年度数据。

### 资产负债表

展示公司总资产、总负债、股东权益、资产负债率等资产状况。

### 现金流量表

展示经营活动、投资活动、筹资活动的现金流入流出情况。

### 核心财务指标

| 类别 | 主要指标 |
|------|---------|
| 盈利能力 | 每股收益（EPS）、净利润、毛利率、净利率、ROE、ROA |
| 估值指标 | 市盈率（PE TTM/动态/静态）、市净率（PB）、市销率（PS） |
| 偿债能力 | 资产负债率、流动比率、速动比率 |
| 成长能力 | 营收增长率、净利润增长率 |

详细字段解释请参考 [行情数据字段介绍](行情数据字段介绍.md)。

## 股东信息

- 股本结构：总股本、流通股本
- 主要股东：前十大股东持股比例
- 机构持仓：机构投资者持仓变动

## 分红信息

- 历史分红记录：派息日期、每股股息金额、股息率
- 分红方式：现金派息、送股、选股选息
- 除权除息日、登记日、派息日等关键日期

查看路径：个股详情页 - 全景 - 百科 - 分红

## 分析师评级

- 目标价：分析师给出的目标价格区间
- 评级分布：买入、增持、中性、减持、卖出的数量分布
- 评级变动：近期分析师评级调整情况

## 公司公告与新闻

- 公司公告：财报发布、公司行动、重大事项公告
- 相关新闻：与该公司相关的市场新闻和资讯
- 社区讨论：投资者社区中关于该股票的讨论

## PortAI 智能助手

长桥 App 内置 PortAI 智能信息助手，可通过对话方式获取个股相关信息，包括财务分析、行业对比、公司行动解读等，帮助高效理解投资信息。
