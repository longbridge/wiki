# ETF行情与全景

ETF（交易所交易基金）详情页分为多个标签，其中**行情**和**全景**是最常用的两个核心标签，分别提供实时报价数据和基金基本面信息。

入口：长桥 App - 市场/自选 - 点击 ETF 标的进入详情页

## 行情 Tab

### 基本行情数据

页面顶部展示 ETF 的核心报价信息：

| 字段 | 说明 |
|------|------|
| 最新价 / 涨跌幅 | 当前成交价及较昨收的涨跌幅 |
| 昨收 | 上一交易日收盘价 |
| 今开 | 当日开盘价 |
| 最高 / 最低 | 当日最高价和最低价 |
| 成交量 | 当日累计成交股数 |
| 成交额 | 当日累计成交金额 |
| 振幅 | （最高价 - 最低价）/ 昨收价 |
| 换手率 | 成交量 / 流通股本 |
| 量比 | 当日成交量与近期均量的比值 |
| 委比 | （买盘委托量 - 卖盘委托量）/ 总委托量，反映买卖力度对比 |
| 均价 | 当日成交均价（成交额 / 成交量） |
| 总市值 / 流通市值 | 总股本 / 流通股本对应市值 |
| 总股本 / 流通股本 | ETF 总份额和可流通份额 |
| 每手 | 最小交易单位 |
| 货币 | 计价货币（如 USD、HKD） |
| 52 周高 / 52 周低 | 近 52 周最高价和最低价 |
| 每股净资产 | 基金单位净值 |
| 每股收益 | EPS（适用于有分红的 ETF） |
| 市盈率 | PE（TTM / 动态 / 静态可切换） |
| 市净率 | PB |
| 股息 / 股息率 | 近期每股分红金额及对应股息率 |

### 盘前 / 盘后交易

支持美股盘前和盘后延长时段行情，顶部显示当前交易时段状态（如「盘前交易中 美东时间」），盘前价格和涨跌幅单独展示。

### BBO（最优买卖报价）

显示当前买一价和卖一价（Best Bid & Offer），以及对应的委托量，帮助判断即时买卖报价水平。

### 深度摆盘

- 默认显示买卖双方各 5 档报价和委托量
- 订阅 **Nasdaq TotalView** 后可查看 60 档深度摆盘，获取更完整的市场深度数据
- 买盘（绿色）/ 卖盘（红色）分列展示，直观显示多空力量分布

### K 线图表

支持多周期切换：

- **分时图**：1 分钟实时走势，含成交量
- **日 K / 周 K / 月 K / 年 K**：不同周期蜡烛图
- **5 日分时**：近 5 个交易日连续分时走势

内置常用技术指标：
- **MACD**：趋势与动量指标
- **KDJ**：超买超卖判断
- **RSI**：相对强弱指数
- **VOL**：成交量柱状图

### 资金流向

展示 ETF 的资金流入流出情况：

- **实时资金流向**：当日按大单 / 中单 / 小单分类的资金流入和流出金额（单位：万元）
- **历史资金流向**：近期多日资金流向趋势，判断主力资金动向

### 做空数据

展示 ETF 的做空相关统计数据，帮助了解市场空头仓位情况。

### 成交统计

按买卖方向和单笔成交规模统计当日成交分布情况。

### 大家也在关注

与当前 ETF 相关的其他标的（同行业或关联板块），显示各标的当日涨跌幅，便于横向对比。

### 相关资讯

行情页底部展示与该 ETF 相关的市场资讯和新闻，来源包括多家财经媒体。数据来自第三方数据源，仅供参考，不构成投资建议。

## 全景 Tab

全景标签提供 ETF 的基本面和分析数据，分为以下几个模块。

入口：个股详情页 - 全景

### 概况

- **基金简介**：ETF 的投资目标、跟踪指数、基金规模、成立时间等基本信息
- **营收构成 / 业绩构成**：主要持仓行业或地区分布（按占比展示）
- **关注度排名**：在同类标的中的用户关注度排名（如「关注度 Top 26」）
- **市值排名**：在所属行业或板块中的市值排名（如「市值排名 17/92」）

### 股价趋势

展示 ETF 在不同时间维度下的涨跌幅表现：

| 时间段 | 说明 |
|--------|------|
| 当日 | 今日涨跌幅 |
| 1 周 | 近 7 日涨跌幅 |
| 1 月 | 近 30 日涨跌幅 |
| 6 月 | 近 6 个月涨跌幅 |
| 今年 | 年初至今涨跌幅 |
| 1 年 / 3 年 | 近 1 年 / 3 年涨跌幅 |
| 全部 | 上市以来总涨跌幅 |

同时显示**社区看涨比例**（如「看涨 84%」）和关注人数及表态条数。

### 股票估值

对 ETF 进行估值分析，提供三项核心估值指标：

- **市盈率（PE）**：当前 PE 与历史区间、行业中位数的对比
- **市净率（PB）**：当前 PB 与历史区间、行业中位数的对比
- **市销率（PS）**：适合评估前期投入大、盈利周期慢的基金

每项指标展示：当前值、合理区间、历史最大 / 最小值、平均值、中位值、行业中位数及同行业排名。

### 财务评分

根据最新财报综合评估基金财务状况，提供量化评分和描述（如「财务状况一般，环比有所恶化」），可与同行比较。

### 成分股 / 持仓

> 该模块展示 ETF 的持仓明细。

- **成分股列表**：ETF 跟踪指数的成分股名称、权重占比
- **前十大持仓**：持仓股票名称及持仓比例

### 分红

历史分红记录，包含：

- 除权除息日 / 登记日 / 派息日
- 每股（每份额）分红金额
- 分红频率（年度 / 季度 / 月度）

### 机构观点 / 分析师评级

汇总机构分析师对该 ETF 或其跟踪标的的评级意见：

- 评级分布：强力推荐 / 买入 / 持有 / 跑输大盘 / 无意见
- 目标价区间

### 股东信息

- **主要股东**：持股比例最高的机构或个人股东列表，含持股数量、持股比例、持股变动及披露时间
- **基金持仓**：哪些 ETF 或基金持有该标的（如 ARK Space Exploration ETF、Roundhill Space ETF 等），显示基金名称、持仓占比和更新时间

### 利益相关者 / 投资关系

提供公司（或基金管理人）的投资者关系信息。
