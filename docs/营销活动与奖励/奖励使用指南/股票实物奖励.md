# 股票与实物奖励

## 股票奖励

### 长桥香港

留存门槛达标后，「我的」页面将展示领取入口，请在 **60 个自然日**内前往领取。

领取后请前往「奖励纪录」使用，系统将在当日发放股票至账户。

### 长桥新加坡

股票将在达标后的 **5 个工作日**内，以股票卡形式发放至「长桥 App - 我的 - 我的卡券」。

请在股票卡有效期内完成查看和兑换。未在有效期内发起兑换，奖励将过期失效，不予补发。

发起兑换后，股票将在 **10 个工作日**内发放至账户。

## 实物奖励

### 长桥香港

留存门槛达标后，「我的」页面将展示领取入口，请在 **60 个自然日**内前往领取。

领取后请前往「奖励纪录」使用，并在获得奖励的 **15 个自然日**内填写收货信息。

完善收货信息后的 **30 个工作日**内安排采购邮寄，邮寄仅限香港地址。

过期未填写收货信息视为放弃奖励，不予补发。
