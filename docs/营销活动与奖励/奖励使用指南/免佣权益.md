# 免佣权益

## 发放时间

免佣权益达标后，将在 1-3 个工作日内发放至长桥 App「我的卡券」，次日起正式生效。

## 港股终生免佣

免佣范围包括港股证券账户发生证券交易买卖时产生的佣金费用。

以下费用不在免佣范围内：
- 暗盘交易佣金
- 平台使用费
- 交易征费
- 交易费
- 交收费
- 印花税

如需查看完整费率详情，可前往长桥 App - 左上角头像 - 我的费率。

## 美股终生免佣

免佣范围包括美股证券账户发生证券交易买卖时产生的佣金费用。

以下费用不在免佣范围内：
- 暗盘交易佣金
- 平台使用费
- 交易征费
- 交易费
- 交收费
- 印花税

如需查看完整费率详情，可前往长桥 App - 我的 - 我的费率。

## 注意事项

- 港美股终生免佣不适用于新股暗盘、美股期权等交易
